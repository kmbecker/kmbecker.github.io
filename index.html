Liri

LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data. Primarily used NodeJS server application, Twitter API, Spotify API, and OMDB API for this one.

For this project the goal was t use multiple APIs in order to compile information from many different data sources.
